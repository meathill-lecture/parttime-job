浏览器的工作机制

从你输入地址到网页渲染完成，中间经历了什么？

1. 浏览器分析地址，解析域名

关键词：域名，IP，DNS

2. 浏览器包装请求，向 IP 发起

关键词：HTTP header

3. 通过层层转发，最终到达服务器

关键词：网络，traceroute

4. 服务器将请求转发给处理程序

关键词：nginx/apache，php-fpm，node.js

5. 服务器处理请求

  1. 读取缓存
  2. 读取数据库
  3. 生成响应体
  4. 输出响应体

6. 浏览器下载响应体

  1. 边下载边解析边渲染
  2. 加载完 里的样式开始渲染
  3. 同一域名最多发起3个连接

7. 执行 JS 加载更多内容

关键词：单页应用（SPA），路由

自由职业真的好么？

答案：不好

  1. 没有社保
  2. 无法保证工作量
  3. 无法保证收入
  4. 缺少延续性
  5. 收入总量低

接私活一定要谨慎

1. 正职工作绝对优先

  1. 主要收入来源
  2. 跟公司一起成长的机会
  3. 简历的说服力

2. 要能提升自己的能力

  1. 宁可做不挣钱的 side project
  2. 尝试工作中需要但不太接触到的技术
  3. 使用新技术提升效率

3. 积累成功案例

  1. 比如做网站……
  2. 结成松散的联盟
  3. 勤总结勤归纳记笔记

4. 要有合适的财务回报

  1. 不理解技术价值的老板多半是坑
  2. 日薪x1.25为宜
  3. 商量好付费方式，比如 442
  4. 可能要考虑税点

Q&A

问：接私单的时候要多少钱合适？

答：

  1. 先评估工作量
  2. 计算你的日薪（月薪/21.5），乘一个系数
  3. 乘工作量，工作量x2

问：我后端技术不太好，怎么办？

答：最好找技术水平好的朋友合作，亦可在平台上找人合作。建议不要自己硬上。

问：那我找我同事如何？

答：找同事未必是个好的选择，除非你跟他她关系非常好。

问：跟人合作怎么分成？

答：这个要具体人和事具体聊了，每个人的要求和想法都不一样，很难讲。不要担心，谈不拢不合作也行，买卖不成仁义在。

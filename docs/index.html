浏览器的工作机制

从你输入地址到网页渲染完成，中间经历了什么？

1. 浏览器分析地址，解析域名

关键词：域名，IP，DNS

2. 浏览器包装请求，向 IP 发起

关键词：HTTP header

3. 通过层层转发，最终到达服务器

关键词：网络，traceroute

4. 服务器将请求转发给处理程序

关键词：nginx/apache，php-fpm，node.js

5. 处理请求

  1. 读取缓存
  2. 读取数据库
  3. 生成响应体
  4. 输出响应体

6. 浏览器下载响应体

  1. 边下载边解析边渲染
  2. 加载完 里的样式开始渲染
  3. 提升首屏加载速度

7. 执行 JS 加载更多内容

关键词：单页应用（SPA），路由

1. 正职工作绝对优先

  1. 主要收入来源
  2. 跟公司一起成长的机会
  3. 简历的说服力

2. 要能提升自己的能力

  1. 宁可做不挣钱的 side project
  2. 尝试工作中需要但不太接触到的技术
  3. 使用新技术提升效率

3. 积累成功案例

  1. 比如做网站……
  2. 结成松散的联盟
  3. 勤总结勤归纳记笔记

  1. 不理解技术价值的老板多半是坑
